# 🎵 MelodyStream

A modern music-inspired authentication web application built using **HTML, CSS, JavaScript, PHP, and XML**. MelodyStream provides a smooth and interactive user experience with animated login/signup panels, session-based authentication, glassmorphism UI, and parallax background effects.

---

## ✨ Features

- 🔐 User Login & Signup System  
- 👤 Session-Based Authentication  
- 📂 XML-Based User Data Storage  
- 🎨 Modern Glassmorphism UI  
- 🌌 Interactive Parallax Background  
- 🔄 Sliding Login/Signup Animation  
- 📱 Fully Responsive Design  
- ⚡ Error & Success Validation Messages  
- 👤 Profile Popup with Logout Option  

---

## 🛠️ Technologies Used

| Frontend | Backend | Storage |
|----------|----------|----------|
| HTML5 | PHP | XML |
| CSS3 | Sessions | File Storage |
| JavaScript | Authentication | User Data |

---

## 🚀 How It Works

1. User signs up using email and password  
2. User data is stored in `users.xml`  
3. Login credentials are verified using PHP  
4. Session starts after successful login  
5. User is redirected to the dashboard  
6. Profile section displays user information  
7. Logout destroys the session securely

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/MelodyStream.git
```

### 2️⃣ Move Project to Local Server

Place the folder inside:

- `htdocs` (XAMPP)
- `www` (WAMP)

### 3️⃣ Start Apache Server

Open:

```bash
http://localhost/MelodyStream
```
